== Description ==

In current system in wordpress user that type is contributor can write comment.But if site admin want to block witting comment for contributor it possible by activate “block-comment” plugin.
For more information about the plugin please see: http://www.krishe.com/en/block-comment

== Installation ==

1. Upload `block-comment` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
